# 线条小狗官网部署指南

## 方案选择

为了让所有人都能访问您的网站,我推荐使用 **Vercel** 部署,原因如下:

- ✅ **完全免费** (个人项目)
- ✅ **自动HTTPS** (安全访问)
- ✅ **全球CDN** (访问速度快)
- ✅ **自动部署** (推送代码即部署)
- ✅ **零配置** (专为Next.js优化)

## 部署步骤

### 方案一:使用Vercel部署(推荐)

#### 1. 准备Git仓库

首先需要将项目上传到GitHub:

```bash
# 在项目目录执行
cd e:\Antigravity\code\project1

# 初始化Git仓库(如果还没有)
git init

# 添加所有文件
git add .

# 提交
git commit -m "Initial commit: 线条小狗毛绒玩具官网"
```

然后:
1. 访问 [GitHub.com](https://github.com)
2. 登录或注册账号
3. 点击右上角 "+" → "New repository"
4. 仓库名称: `line-dog-toys-website`
5. 选择 "Public" (公开)
6. 点击 "Create repository"

将代码推送到GitHub:
```bash
# 添加远程仓库(替换YOUR_USERNAME为你的GitHub用户名)
git remote add origin https://github.com/YOUR_USERNAME/line-dog-toys-website.git

# 推送代码
git branch -M main
git push -u origin main
```

#### 2. 部署到Vercel

1. **访问Vercel**
   - 打开 [vercel.com](https://vercel.com)
   - 点击 "Sign Up" 注册(可以用GitHub账号登录)

2. **导入项目**
   - 登录后点击 "Add New..." → "Project"
   - 选择 "Import Git Repository"
   - 授权Vercel访问你的GitHub
   - 选择 `line-dog-toys-website` 仓库

3. **配置项目**
   - Project Name: `line-dog-toys` (或自定义)
   - Framework Preset: Next.js (自动检测)
   - Root Directory: `./`
   - 保持其他默认设置

4. **部署**
   - 点击 "Deploy" 按钮
   - 等待2-3分钟完成部署

5. **获取网址**
   - 部署完成后,会得到一个网址,例如:
     ```
     https://line-dog-toys.vercel.app
     ```
   - 这个网址全球任何人都可以访问!

#### 3. 自定义域名(可选)

如果您有自己的域名(如 `www.linedogtoys.com`):

1. 在Vercel项目设置中点击 "Domains"
2. 添加你的域名
3. 按照提示在域名服务商处添加DNS记录
4. 等待DNS生效(通常5-30分钟)

### 方案二:使用Netlify部署(备选)

如果Vercel不可用,可以使用Netlify:

1. 访问 [netlify.com](https://netlify.com)
2. 注册并登录
3. 点击 "Add new site" → "Import an existing project"
4. 连接GitHub并选择仓库
5. Build settings:
   - Build command: `npm run build`
   - Publish directory: `.next`
6. 点击 "Deploy site"

### 方案三:使用国内平台

如果需要在国内部署,可以考虑:

1. **阿里云/腾讯云**
   - 需要购买服务器
   - 需要备案域名
   - 成本较高,适合商业项目

2. **Cloudflare Pages**
   - 免费
   - 全球CDN
   - 类似Vercel的使用体验
