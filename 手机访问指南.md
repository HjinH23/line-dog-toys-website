## 如何在手机上访问

### 步骤 1: 重启开发服务器

由于修改了 `package.json`,需要重启开发服务器:

1. 在当前运行 `npm run dev` 的终端按 `Ctrl+C` 停止服务器
2. 重新运行:
   ```bash
   npm run dev
   ```

### 步骤 2: 确认服务器启动

启动后,您应该看到类似这样的输出:
```
▲ Next.js 15.1.3
- Local:        http://localhost:3000
- Network:      http://192.168.4.170:3000
```

### 步骤 3: 在手机上访问

**重要**: 确保手机和电脑连接到同一个WiFi网络!

在手机浏览器中输入:
```
http://192.168.4.170:3000
```

> **注意**: 您的电脑IP地址是 `192.168.4.170`,如果IP地址变化,请使用新的IP地址。

### 步骤 4: 检查防火墙

如果手机无法访问,可能是Windows防火墙阻止了连接。请执行:

1. 打开 Windows 设置 → 更新和安全 → Windows 安全中心 → 防火墙和网络保护
2. 点击"允许应用通过防火墙"
3. 找到 Node.js 或允许端口 3000

或者临时关闭防火墙测试(不推荐长期使用)。

## 常见问题

**Q: 手机显示"无法访问此网站"?**
A: 
1. 确认手机和电脑在同一WiFi
2. 确认开发服务器正在运行
3. 检查Windows防火墙设置
4. 尝试重启开发服务器

**Q: 页面加载很慢?**
A: 
1. 这是开发模式,首次加载会较慢
2. 确保WiFi信号良好
3. 生产环境会快很多

**Q: IP地址变了怎么办?**
A: 
1. 重新运行 `ipconfig` 查看新IP
2. 使用新IP访问,格式: `http://新IP:3000`
